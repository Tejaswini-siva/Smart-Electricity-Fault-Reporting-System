The Smart Electricity Fault Reporting System is a web-based application designed to efficiently report, track, and manage electricity-related faults. It helps users notify authorities about issues like power outages, line faults, and electrical hazards, improving response time and service quality
Features:

1. Report electricity faults with location details

2. Categorize faults (power cut, line damage, transformer issues, etc.)

3. View fault reports with severity and date

4. Real-time data handling using REST APIs

5. Simple and user-friendly interface

6. Backend integration for storing and retrieving data
